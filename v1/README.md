# Weather Analyzer (Version 1)

## ✨ Key Highlights
- 🔁 Program flow was controlled by a single **while True loop**.
- 📑 Menu choices were handled using **if/elif statements** (1, 2, 3, etc.).
- 📊 Summary and visualization options were triggered directly inside the loop.
- 🖥️ All logic lived together in one place, making it simple to follow.
- 🎯 Focus was on building a working menu-driven program quickly.
